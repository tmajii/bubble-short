Penjelasan Singkat Flowchart
-Program dimulai dan menerima data array.
-Panjang array dihitung dan disimpan dalam variabel n.
-Perulangan luar (i) mengatur jumlah iterasi pengurutan.
-Perulangan dalam (j) membandingkan elemen yang bersebelahan.
-Jika elemen kiri lebih besar, maka dilakukan penukaran.
-Setiap iterasi membuat elemen terbesar berpindah ke posisi akhir.
-Setelah semua proses selesai, array menjadi terurut.
-Program berakhir.
